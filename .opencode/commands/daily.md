---
description: Buat atau buka daily note hari ini
---

Buat daily note hari ini di `docs-braille/daily/{YYYY-MM-DD}.md` menggunakan template dari `.opencode/templates/daily.md`. Ganti `{DATE}` dengan tanggal hari ini. Jika file sudah ada, baca dan lanjutkan konten yang ada.

Setelah membaca daily note, pahami prioritas dan konteks sebelum mulai bekerja. Di akhir sesi, tulis session log ke bagian "Session Log" di daily note.

Pastikan daily note memiliki link `[[MEMORY]]` di footer (template sudah include). Ini penting untuk Obsidian graph agar daily notes terhubung ke ringkasan project.
